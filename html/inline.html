# 关于行内元素

行内元素只占本身大小, 可以跟其他共处一行 但并不能设置宽高(除了这几位替换元素:img、audio、video、canvas)等
ps: 然而img、audio、video、canvas其实是行内块元素, 只是w3c默认把行内块元素归为行内元素。

  • b

  • sup

# 关于行内元素布局的特点

当行内元素并排紧贴时, 行内元素之间会有个缝隙解决方案：给其父元素的 font-size：0 , 若元素需要设置 font-size 再在自行设置。

# b

  • 表示的文本从正常文本不同文体的跨度，没有传送任何特殊的重要性或相关性。
  • 一般用来表示只想不同但没有强调的意思的内容。
  • 以粗体呈现。
  • 仅当没有更合适的语义元素时才使用此元素。

# del

  • 定义删除文本。
  • 通常以删除线为表现形式。
   <!-- 实现打折的效果 -->
   <del>&yen;50</del>
   <!-- 无法使用del的情况下 -->
   <span style="text-decoration: line-through;">&yen;50</span>

# em

  • 表示需要强调的文本。
  • 强调的程度比strong弱.
  • 斜体的形式呈现。

# i

  • 一般技术术语，外语短语，或虚构的人物想法，没有什么语义。
  • 与em标签一样以斜体的形式呈现，但与em标签的语义不同。
  • 仅当没有更合适的语义元素时才使用此元素。

# mark

它可以在显示搜索结果的页面中使用，以突出搜索词的每个实例

# span

span是个无色无味的容器，跟div相似但div是块级元素而span是行内元素。
主要用于将元素分组添加class或id属性，仅当没有语义元素时才使用它。

    <p><span>Some text</span></p>
    <li>
        <span>
            <a href="portfolio.html" target="_blank">See my portfolio</a>
        </span>
    </li>

# strong

表示需要强调的文本。
强调的程度比em强，如果em是大叫,那么strong就是尖叫。
以粗体形式呈现。 通常用于”警告！这非常危险“这一类的文本。
